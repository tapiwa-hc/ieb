This page displays all the notices on the system which have been approved

(If you are a staff member and it's one of your own notices, or you are an admin)

In each row, you can edit the title and description columns by double-clicking the text

  • Click "Keep" to store changes
  • Click "Discard" to revert changes

(If you are a staff member or admin)

Once satisfied with changes made, click the "Save Changes" button in the bottom right corner of the page to write changes to the database. The table will update if successful, otherwise you will be informed of an error.
